# Prompts

This folder contains prompt templates and examples used in the COCA project.

## Files

- [`gpt4o_evaluation_prompt.md`](gpt4o_evaluation_prompt.md): GPT-4o evaluation prompt for judging answer correctness.
- [`gpt4o_output_examples.md`](gpt4o_output_examples.md): example JSON outputs for GPT-4o evaluation.
- [`coca_prompts.md`](coca_prompts.md): critic and actor prompts used in the COCA method.
